# Sample Test Portfolios for Structura 2.0

## CSV Format
The app expects CSV with these columns:
- symbol: Stock symbol (e.g., RELIANCE, TCS, INFY)
- quantity: Number of shares
- avgPrice: Average buy price per share

## Test Portfolio 1: Balanced Portfolio (5 stocks)

```csv
symbol,quantity,avgPrice
RELIANCE,10,2500
TCS,5,3800
INFY,15,1500
HDFCBANK,8,1600
ITC,50,450
```
